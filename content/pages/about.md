# Sobre mí

¡Hola! Soy **Álvaro Quero**, desarrollador full stack apasionado por crear soluciones web modernas y eficientes.

## ¿Quién soy?

Soy un desarrollador con experiencia en el desarrollo de aplicaciones web utilizando tecnologías modernas como React, Next.js, Node.js, y ahora Astro. Me encanta aprender nuevas tecnologías y aplicarlas para resolver problemas reales.

## Mis habilidades

### Frontend
- React.js / Next.js / Astro
- TypeScript / JavaScript
- Tailwind CSS / CSS Modules
- Responsive Design
- Accesibilidad Web

### Backend
- Node.js / Express
- APIs RESTful
- Bases de datos SQL y NoSQL
- Autenticación y Autorización

### Herramientas y Metodologías
- Git / GitHub
- CI/CD
- Metodologías Ágiles
- Testing (Jest, React Testing Library)

## ¿Qué me motiva?

Me apasiona crear experiencias web que sean:
- **Rápidas**: Optimizadas para el mejor rendimiento
- **Accesibles**: Para que todos puedan usar mis aplicaciones
- **Intuitivas**: Con diseños centrados en el usuario
- **Mantenibles**: Código limpio y bien documentado
