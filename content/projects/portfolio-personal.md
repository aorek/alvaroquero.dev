# Portfolio Personal

Mi sitio web personal donde comparto mis proyectos, blog posts y experiencia profesional.

## Características

- 🚀 **Ultra rápido**: Generado estáticamente con Astro
- 📝 **Blog integrado**: Sistema de blog con Markdown
- 🎨 **Diseño moderno**: UI limpia y responsive con Tailwind CSS
- 🌗 **Dark mode**: Soporte para tema oscuro
- 📱 **Responsive**: Funciona perfecto en todos los dispositivos
- ♿ **Accesible**: Diseñado siguiendo principios de accesibilidad

## Tecnologías utilizadas

### Frontend
- **Astro**: Framework para sitios web rápidos
- **TypeScript**: Para type safety
- **Tailwind CSS**: Estilos utility-first
- **React**: Para componentes interactivos (Islands)

### Content Management
- **Content Collections**: Sistema de gestión de contenido de Astro
- **Markdown/MDX**: Para escribir posts y proyectos

### Deployment
- **Vercel**: Hosting y CI/CD

## Desafíos

1. **Migración desde Next.js**: Adaptar componentes y routing
2. **Dark mode**: Implementar sin flash de contenido
3. **PDF Export**: Generar CV en PDF desde el navegador

## Aprendizajes

- Content Collections de Astro son increíbles para gestionar contenido
- Astro Islands permite tener React solo donde se necesita
- Tailwind CSS acelera el desarrollo de UI
